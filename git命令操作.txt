创建版本库
git init
git add filename添加文件
git commit -m '注释'
git status是否有文件未提交
git diff filename查看具体修改的内容
版本回退
git log
git log --pretty=online
回退原来的版本：
git reset --hard HEAD^
git reset --hard HEAD^^
git reset --hard HEAD^100
cat x.txt查看文件内容
恢复到最新的版本：
git reset --hard 版本号
如果忘了版本号：
git reflog
git reset --hard 版本号
丢弃工作区的修改：
git checkout -- x.txt
git restore x.txt
先添加到暂存区中，然后修改的情况，撤销之后回到添加到暂存区后的状态：
git add x.txt
git checkout -- x.txt
删除文件：
rm b.txt
在未commit之前，恢复删掉的文件：
git checkout -- b.txt
远程仓库：
ssh-keygen -t rsa -C "1750371110@qq.com"
add ssh keys
添加远程库：
git remote add origin https://github.com/tugenhua0707/testgit.git
git push -u origin master         (由于远程库是空的，所以第一次推送master分支的时候要加上-u)
从现在起，只要本地做了提交，然后就可以通过如下命令将本地分支的修改推送到github上了
